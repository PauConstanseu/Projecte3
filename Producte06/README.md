# Introducció a la tasca: Vídeo sobre els fonaments del servei DNS

En aquesta activitat hem treballat per crear un vídeo explicatiu sobre els fonaments del servei **DNS (Domain Name System)**. L’objectiu principal era comprendre com funciona aquest sistema essencial per a la navegació a Internet i transmetre aquesta informació de manera clara i visual. El DNS és el mecanisme que tradueix noms de domini fàcilment recordables (com *www.exemple.com*) en adreces IP que els ordinadors utilitzen per comunicar-se. Sense aquest servei, la connexió entre usuaris i recursos digitals seria molt més complexa.

El vídeo pretén mostrar:
- **Què és el DNS i per què és important**.
- **Com es realitza la resolució de noms**.
- **Els components principals del sistema** (servidors autoritatius, recursors, etc.).
- **Exemples pràctics** per entendre el procés.

Amb aquesta tasca hem pogut reforçar conceptes teòrics i aplicar-los en un format audiovisual que facilita l’aprenentatge.
